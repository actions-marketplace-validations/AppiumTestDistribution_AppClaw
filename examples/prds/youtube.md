# YouTube Mobile App - PRD

## Overview

YouTube is a video streaming application where users can discover, watch, and interact with videos.

## Core Features

### Video Search

- Users can search for videos using the search icon
- Search results show video thumbnails, titles, channel names, and view counts
- Users can filter search results by relevance, upload date, view count

### Video Playback

- Users can tap on a video to play it
- Video player supports play/pause, seek, fullscreen
- Videos show title, channel name, description, like/dislike counts

### Home Feed

- Home screen shows recommended videos
- Users can scroll through the feed
- Each video card shows thumbnail, title, channel, views, and upload time

### Shorts

- Short-form vertical videos accessible from the bottom navigation
- Users can swipe up/down to browse shorts

### Subscriptions

- Users can view their subscribed channels
- Subscription feed shows latest videos from subscribed channels

### Library

- Users can access watch history, playlists, and downloads
- Library section accessible from bottom navigation
